## 技术栈

nodejs + express + mongodb + mongoose + es6/7


## 项目运行


```
git clone https://github.com/ymqq/crm-middle-server  

cd node-elm

npm install

npm run dev (需先开启mongodb)

访问: http://localhost:8001

```
